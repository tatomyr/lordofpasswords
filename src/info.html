Why use Password Generator?

It's safe to have a different password for each site or another service. And it's also good to have passwords that hard to guess but easy to remember.

That's where the Password Generator comes in handy. You should only remember your masterpassword. Also, you have to specify a memorable name for a site or service (for this site it probably could be 'TODO: passwordity/fy'). Then the app does the rest - mixes them both together, and ensures that you'll get the proper password format (e. g. with the required number of different types of characters).

You can access the Password Generator on desktop, on a tablet or on your mobile. You can even kind of install it!

Then if you know which service you're trying to get access to, and remember your single masterpassword, you can always have your key using Password Generator.

Is it safe to use Password Generator?

The short answer is yes, it is. And that's why.

Using this application is completely anonymous. We don't track your identity, so we don't know which user has issued a password.

Moreover, we don't send your data over the internet, neither store it, except for password length. This is storing locally in the sake of user's convenience, so you don't need to enter the number each time. You can go completely offline and your app will continue working because we don't rely on any kind of database. So no one could crack your sensitive data, because it isn't stored anywhere but inside your head.

To ensure that someone else can't access your data, we don't depend on any third-party library that may change over time and possibly bring some malicious instructions.

Also, we prevent any call over the internet from this app. So even if some malicious extension did read your personal data, they wouldn't have sent them anywhere.

All the application code is available on Gitlab. Also, we don't minify nor uglify the app code so you can check it out using developer tools.
